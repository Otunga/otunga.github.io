Module nut.class

This module contains all the functions that handle classes.

Classes are different from factions, though all classes belong to a faction. They allow you to create different and specialized jobs and with that, custom salaries, models and other functions.

Functions

nut.class.loadFromDir(directory)
Loads data from the classes directory. Loads all classes' data from the 'classes' directory inside your schema folder.

Parameters:

  • directory string the path to the classes directory.

Returns:

    nothing.
nut.class.canBe(client, class)
Determines whether a player is allowed to join a certain class. The function checks if the client is eligible to join the class. Returns true if the player is allowed to join the class and false otherwise, along with the reason as to why the player is unable to join.

Parameters:

  • client player a player.
  • class a number.

Returns:

    a boolean value.

Usage:

    nut.class.canBe(client, 5)
nut.class.get(identifier)
Gets the class with the given identifier. The function returns the table of a specific class. This table is inside nut.class.list.

Parameters:

  • identifier a number.

Returns:

    the class table.
nut.class.getPlayers(class)
Gets all the players in a given class. The function returns a table with all the players in the given class.

Parameters:

  • class a number.

Returns:

    the table with the players.
