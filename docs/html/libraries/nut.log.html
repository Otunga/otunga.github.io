Module nut.log

This module contains all the functions that handle logging.

NutScript has a logging system. It logs core related actions and custom actions that you wish to be logged. The framework seperates all logs into different categories.

Enumerations:

FLAG_NORMAL = 0
FLAG_SUCCESS = 1
FLAG_WARNING = 2
FLAG_DANGER = 3
FLAG_SERVER = 4
FLAG_DEV = 5

Whenever you wish to log a command or something else, you use nut.log.add function.

nut.log.loadTables()
Creates the nutscript logs directory under DATA.

Returns:

    nothing.
nut.log.addType(logType, func)
Allows you to add a different logs section. Adds the new logs category to nut.log.types.

Parameters:

  • logType string a log's category.
  • func a function.

Returns:

    nothing.
nut.log.getString(client, logType)
Gets a specific log. If there is no logType in nut.log.types table, the function return "-1" value.

Parameters:

  • client player a player.
  • logType string a log's category.

Returns:

    a string.
nut.log.addRaw(logString, flag)
Sends a log to all the staff members online with a specific flag. The function also appends the log to a logfile, under the logs directory.

Parameters:

  • logString string a log entry.
  • flag a number.

Returns:

    nothing.

  • client player a player.

Returns:

    nothing.
