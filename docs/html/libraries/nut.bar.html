Module nut.bar

This module contains all the functions that handle bars.

This library is CLIENTSIDE library that allows you register and manage new bars on top left corner.

Functions

nut.bar.get(identifier) This function returns the bar with the specified identifier.
nut.bar.add(getValue, color, priority, identifier) Registers a new bar.
nut.bar.remove(identifier) Removes a bar with the specified identifier.
nut.bar.draw(x, y, w, h, value, color) Draws a bar on the screen.
