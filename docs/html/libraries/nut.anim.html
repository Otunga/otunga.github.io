nut.anim.setModelClass(model, class)
A shared function to group a certain model with a specific animations.
Class is a string that can be: "citizenmale", "citizenfemale", "metrocop", "overwatch", "vort", "player", "zombie" and "fastZombie". Note that the last three are only for regular player models, not NPC models.

Parameters:

  • model string the model's path.
  • class string the animation class.

Returns:

    error if it is not a valid animation class.
nut.anim.getModelClass(model)
Returns the class for the model specified. If one has not been set previously, it will default to "player" if the model contains "/player", otherwise the citizen classes.

Parameters:

  • model string the model's path.

Returns:

    the class for the model specified.
