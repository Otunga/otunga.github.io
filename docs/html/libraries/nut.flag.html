Module nut.flag

This module contains all the functions that handle flags.

Flags allow you to restrict certain features to all those who have access to that specific flag. For example, you can make specific items available for purchase to only those who have a specific flag.

NutScript already has default flags:

  • p - grants access to the Physgun;
  • t - grants access to the Toolgun;
  • c - grants access to spawn chairs;
  • C - grants access to spawn vehicles;
  • r - grants access to spawn ragdolls;
  • e - grants access to spawn props;
  • n - grants access to spawn NPCs.

nut.flag.add(flag, desc, callback)
Adds a flag to the flags module. The function adds the flag, description and the callback function if there is one to the nut.flag module.

Parameters:

  • flag string the flag prefix.
  • desc string the flag's description.
  • callback a function.

Returns:

    nothing.

Usage:

    nut.flag.add("m", "access to a restricted business menu tab.")
