Module nut.attribs

This module contains all the functions that handle attributes.

Character attributes are used in roleplay. You can create new attributes on your schema folder by creating a directory called 'attributes' and a file per attribute.

Functions

nut.attribs.loadFromDir(directory) Loads data from the attributes directory.
nut.attribs.setup(client) Sets up the attributes for the client's character.

Loads data from the attributes directory. Loads all attributes' data from the 'attributes' directory inside your schema folder.

Parameters:

  • directory string the path to the attributes directory.

    nothing.
