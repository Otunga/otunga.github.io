Module nut.currency

This module contains all the functions that handle currency.

You can set your gamemode's currency by using nut.currency.set. If no currency is set, the framework will use dollar ("$") as the default currency.

Functions

nut.currency.set(symbol, singular, plural) Sets the gamemode currency, which is going to be used.
nut.currency.get(amount) Gets the gamemode currency that is currently being used.
nut.currency.spawn(pos, amount, angle) Spawns a certain amount of money.


Functions

nut.currency.set(symbol, singular, plural)
Sets the gamemode currency, which is going to be used. This function changes the default values of nut.currency.symbol, nut.currency.singular and nut.currency.plural.

Parameters:

  • symbol string the currency's symbol.
  • singular string singular.
  • plural string plural.

Returns:

    nothing.

Usage:

    nut.currency.set("Â£", "coin", "coins")
nut.currency.get(amount)
Gets the gamemode currency that is currently being used. This functions returns a string with a specific amount along with the symbol and the currency singular or plural.

Parameters:

  • amount a number.

Returns:

    a string.
nut.currency.spawn(pos, amount, angle)
Spawns a certain amount of money. This function creates a money entity on a certain position and angle.

Parameters:

  • pos a vector.
  • amount a number.
  • angle an angle.

Returns:

    the entity that was created.
